# VesperiaTools
Collection of scripts and tools to extract data from Tales of Vesperia Definitive Edition.

## Requirements
- Python 3.6+
- Windows 7 SP1 or newer

## Dependencies
Uses Admiral Curtiss' [HyoutaTools](https://github.com/AdmiralCurtiss/HyoutaTools) to perform the data extraction. Knowledge of compiling Visual Studio project are required to compile HyoutaTools.

Place the compiled executables and DLLs into `hyoutatools` directory at root level.
